# 🚀 Backlog: User Me UI Upgrade Plan (Phase 2-1+)

## 📋 개요
Phase 2-1(MVP)에서는 사용자의 기본 텍스트 정보(이메일, 이름 등)를 조회하는 데 집중했습니다.
다음 단계에서는 **시각적 아이덴티티(프로필 이미지)** 강화, **계정 관리(탈퇴, 비밀번호)** 접근성 확보, 그리고 **사용자 경험(UX) 고도화**를 목표로 합니다.

---

## 1. 프로필 이미지 (Profile Image & Avatar)

### 🖼️ 이미지 업로드 및 표시 (Critical)
- **현재**: `Shadcn Avatar` 컴포넌트를 사용 중이나, 실제 이미지가 없어 `Fallback`(이니셜)만 표시됨.
- **업그레이드**:
  - **DB/API**: `users` 테이블에 `profile_image_url` 필드 추가 및 조회 API 응답에 포함.
  - **Upload UI**: 아바타 클릭 시 파일 선택 창 오픈 -> 이미지 크롭(Crop) -> 서버 전송(`POST /users/image`).
  - **Storage**: AWS S3, Cloudinary 또는 로컬 스토리지에 이미지 저장 로직 연동.

### 🎲 동적 아바타 (Dynamic Placeholder)
- **개선**: 사용자가 이미지를 올리지 않았을 때, 단순 이니셜 대신 `DiceBear` 같은 API를 활용하여 랜덤하지만 예쁜 프로필 아이콘을 자동 생성하여 제공.

---

## 2. 계정 관리 및 보안 (Account & Security)

### 🗑️ 회원 탈퇴 (Delete Account)
- **기능**: 서비스 사용을 중단하고 개인정보를 파기할 권리 제공.
- **UX**:
  - 실수 방지를 위한 **'Danger Zone'** 섹션 분리 (붉은색 테두리/버튼).
  - 탈퇴 시 "정말로 떠나시나요?" 모달 팝업 + **비밀번호 재확인** 절차 필수.

### 🔑 비밀번호 변경 진입점
- **현재**: 비밀번호 재설정은 로그아웃 상태에서만 접근 가능.
- **개선**: 로그인된 상태에서도 **[비밀번호 변경]** 버튼을 통해 `Current Password` -> `New Password` 변경 모달 제공.

---

## 3. 사용자 설정 (Preferences)

### 🌓 다크 모드/라이트 모드 (Theme Toggle)
- **기능**: 시스템 설정과 무관하게 사용자가 원하는 테마를 고정할 수 있는 옵션 제공.
- **구현**: `next-themes` (또는 React Context)를 활용하여 설정 값을 로컬 스토리지에 저장.

### 🌐 언어 설정 (Localization)
- **기능**: 한국어/영어(KO/EN) 전환 토글.
- **확장**: 향후 다국어 서비스 확장을 대비한 `i18n` 설정 UI 배치.

---

## 4. UX 및 편의성 (Micro Interactions)

### 📋 아이디/이메일 복사
- **기능**: 내 이메일이나 `user_id` 옆에 작은 **[복사]** 아이콘 배치.
- **용도**: 고객센터 문의나 디버깅 시 자신의 식별자를 쉽게 복사할 수 있도록 지원.

### 📊 활동 요약 대시보드 (Dashboard)
- **아이디어**: 단순히 정보만 나열하지 않고, "내가 좋아요한 영상 수", "작성한 댓글 수" 등을 뱃지(Badge) 형태의 통계로 보여주어 만족감 부여.

---

## 5. 실행 로드맵 (Roadmap)

| 우선순위 | 작업명 | 설명 | 기술 스택 |
| :--- | :--- | :--- | :--- |
| **Highest** | **프로필 이미지 (User Image)** | 업로드, 조회, 표시 | `Form Data`, `Multer`(Back) |
| **High** | 정보 수정 (Edit Info) | 닉네임, 소개글 수정 | React Hook Form |
| **Mid** | 회원 탈퇴 (Withdraw) | 데이터 삭제 및 로그아웃 | Modal, Delete API |
| **Mid** | 테마 설정 (Theme) | 다크모드 토글 | Shadcn Theme |
| **Low** | 활동 통계 (Stats) | 좋아요/댓글 카운트 | Dashboard UI |
