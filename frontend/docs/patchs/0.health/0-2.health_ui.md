**UI SPEC (Shadcn First)**:
- **Layout**: 화면 중앙 정렬 (`flex items-center justify-center h-screen`).
- **Components**:
    - `<Card>`: 상태 정보를 담을 컨테이너.
    - `<Badge>`: 상태 표시 (Success="live" → Green, Error → Red).
    - `<Button>`: "새로고침" 기능 (React Query `refetch`).
- **State Feedback**:
    - **Loading**: 스켈레톤 UI 또는 "Checking Server Status..." 텍스트.
    - **Error**: "Server Offline" 메시지와 재시도 버튼.
    - **Success**: "System Online" 메시지와 `uptime_ms`, `version` 표시.

**DATA SPEC (Strict snake_case)**:
- **Endpoint**: `GET /healthz`
- **Response DTO (`HealthRes`)**:
    - `status`: string ("live")
    - `uptime_ms`: number
    - `version`: string
- **Using Type**: `src/category/health/types.ts`에서 `HealthRes` import 필수.
