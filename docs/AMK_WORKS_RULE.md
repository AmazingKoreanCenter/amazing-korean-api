# 작업 검증 루프

이 문서가 첨부(@)되면 아래 절차를 반드시 따른다.

## 규칙
- 추측 금지. 모든 답변은 도구(Read, WebSearch, Grep 등)로 확인한 근거 기반
- "확인했다"고 말할 때 어떤 도구로 어떻게 확인했는지 명시
- 기존에 가지고 있던 학습 데이터 기반 답변 금지

## 작업 완료 후 검증
1. 사용자의 원래 요청을 다시 읽는다
2. 요청의 요구 항목을 체크리스트로 나열한다
3. 각 항목에 대해 실제 결과물을 Read로 확인한다
4. 미완료 항목이 있으면 실행한다
5. 3-4를 누락 없을 때까지 반복한다
6. 최종 체크리스트를 사용자에게 보여준다

## 금지 사항
- 검증 루프를 생략하거나 축약하지 않는다
- 체크리스트 항목은 절대 실제 검증 없이 "완료"로 표시하지 않는다
