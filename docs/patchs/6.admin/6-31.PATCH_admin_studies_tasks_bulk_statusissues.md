# 📝 Phase 6-31 Admin Bulk Update Task Status - Issue & Troubleshooting Log

## 1. 컴파일 이슈 (Compilation Issues)

### ⚠️ 이슈 1: 반환 타입 불일치 (`no method named ok_or`)
* **현상:** `repo::find_study_task_by_id_tx` 호출 후 `.ok_or(AppError::NotFound)`를 체이닝하자 컴파일 에러 발생.
* **원인:** 해당 Repository 함수는 `Option<T>`가 아니라, 이미 내부적으로 에러 처리가 완료된 `Result<T>`(성공 시 구조체 직반환)를 반환함. 따라서 `Option`에 사용하는 `ok_or` 메서드가 존재하지 않음.
* **해결:** 불필요한 `.ok_or(AppError::NotFound)` 호출을 제거하고 `.await?` 결과값을 바로 변수에 할당하도록 수정.

### ✅ 검증된 로직의 안전한 재사용 (Safe Reuse)
* **성과:** Phase 6-30에서 `sqlx`의 콤마 이슈(`push_bind_unseparated`)를 완벽하게 해결해 둔 `update_task_status` 함수를 그대로 재사용함.
